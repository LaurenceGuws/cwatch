# Experimental Grid Input Checklist

- [x] Single click: focus + select cell
- [x] Arrow keys: move focus only (selection unchanged)
- [x] Shift+Arrow: extend selection range from anchor
- [x] Ctrl/Cmd+Arrow: jump to edge of data in row/col (uses value-aware edge, all cells populated)
- [x] Home/End: move to row start/end (Shift extends)
- [x] Ctrl+Home/End: move to grid corners (Shift extends)
- [x] PageUp/PageDown: move by chunk (Shift extends)
- [ ] Enter/Shift+Enter: move down/up; commit edit in edit mode
- [x] Ctrl/Cmd+Enter: add focused cell to selection without filling range
- [ ] Tab/Shift+Tab: move right/left; wrap rows
- [x] F2: toggle edit mode
- [x] Enter in nav mode: select focused cell
- [ ] Esc: exit edit/cancel selection changes
- [x] Ctrl+Space: select column (anchor on active cell)
- [x] Shift+Space: select row (anchor on active cell)
- [x] Ctrl+A: select all
- [x] Double click: enter edit mode
- [ ] Mouse drag: marquee selection across cells
- [ ] Context menu on right click/long press: cell/row/col actions
- [ ] Column resize via header drag
- [ ] Fill handle drag to copy
- [ ] Touch tap/double tap: focus/select/edit
- [ ] Touch long press: context menu
- [ ] Touch drag selection + edge scroll
- [ ] Pinch zoom: grid zoom
- [ ] Scroll-to-cell when moving selection
- [ ] Mouse scope: hover/down/up handling and containment
